AI Survival Kit ／ はじめての方へ

AIでアプリを作り始めた、
あなたへの最初の地図。

コードも用語も知らなかった人間が、公開まで辿り着いた記録です。

AIに「アプリを作って」とお願いすると、最初の数分は魔法みたいに進みます。でも、ほとんどの人が同じ場所で固まる。コードの中身ではなく、その手前で。

「ターミナルって、どれ？」「ログって、どこを見るの？」「“デプロイ手順”って書いてあるけど、結局なにをやればいいの？」――この“見えない段差”だけを集めたのが、このキットです。

01なぜ、GitHub なんていう難しそうな場所に置いてあるの？

「リンクを開いたら、英語のファイルがズラーっと並んでて、“何これ…？”ってなった。」

その感覚、まったく正常です。むしろ、それを感じられた時点で、あなたはもう一歩“作る側”に近づいています。

正直に言います。大なり小なり「アプリを作る側」に回ったなら、GitHub は避けて通れない場所です。だから、ここで尻込みして引き返すより、今のうちに、ちょっとだけ触れて慣れておく。それ自体が、立派な第一歩になります。逃げ場所ではなく、通り道として用意しました。

GitHub をひとことで言うなら――
世界中の作り手が、自分の作品や「こうすればもっと良くなる」というアイデアを、無料で持ち寄って置いている“共同の作業場”です。
だから、ここに通えるようになること自体が、「使うだけの人」から「作る側の人」への入口になります。

「じゃあ、ここに置いてあるものは全部すごいアイデアなの？」

いいえ。ここが大事なところ。全部が有能なアイデアばかりじゃありません。
中には、針の穴を通すような――「それ、いつ使うの？」というくらいマニアックすぎる代物も、平気で転がっています（笑）。
だから必要なのは、“全部を取り込もうとしない目”。世界中のアイデアの中から、今の自分に本当に要るものだけを選ぶ。これは、このキットがずっと伝えている考え方そのものです。

AIも、GitHub も同じです。便利なものを“全部”飲み込もうとすると、かえって溺れる。「無いと死ぬもの」と「あると便利なだけのもの」を、自分で選り分ける。その目を持つための練習台として、まず GitHub に触れてみてください。

02なぜ note や有料 note じゃないの？

「優しい入口にしたいなら、もっと読みやすい場所もあるのでは？」と思うかもしれません。それでも GitHub に置いたのには、理由があります。

読みやすくまとめた“商品”として売る形にすると、どうしても「続きは登録／購入で」という導線が生まれがちです。このキットは、そういう囲い込みの真逆を行きたい。売り込みも、登録もなしで、ただ置いてある。必要な人が、必要なところだけ持っていけばいい。GitHub は、それが一番自然にできる場所でした。

03「Wikipedia みたいなもの？」――半分そう、半分まったく逆。

無料で、登録なしで読めて、みんなの力で少しずつ育っていく。その意味では、確かに Wikipedia に近い“公共物”を目指しています。

でも、決定的に違うところが2つあります。

① Wikipedia は「全部載せる」。このキットは「あえて削る」。
百科事典は、できるだけ多くの知識を完全に集めようとします。でもこのキットは逆で、「無いと死ぬもの」だけを残して、便利なだけのものは全部わざと隔離しました。完全であろうとせず、あえて小さくあろうとしています。
② Wikipedia は「正しい知識」を載せる。このキットは「失敗の記録」を載せる。
ここに並ぶのは、きれいな正解集ではありません。「自分はここで詰まった、転んだ、死にかけた」という生の記録です。客観的な百科事典ではなく、一人称の“遭難日記”に近い。だから、読んでいて「あ、それ自分も…」と笑えるはずです。

みんなで育てる公共物なのに、あえて小さく、あえて失敗を晒す。このちぐはぐさが、このキットの個性です。

04そもそも、これは“どんな人”向け？

AIでサイトやアプリを作るとき、大きく2つのタイプに分かれます。

① 用意された枠に、文字や画像を入れて作る人
（BASE や Wix など。コードはほぼ触りません）

② 中身のコードにも手を入れて作る人
（Shopify のテーマ改造や、AI にアプリを作らせる、など）

このキットが効くのは、主に ② の人です。①だけで完結している人には、正直あまり刺さりません。――そこは、正直に。

――ただ、①の人が「もう少し自分でいじってみたい」と ② に踏み出す。その時には、最初の一歩の地図になれると思います。

05開いたら、まずこれだけ。

難しいことは要りません。最初にやることは、たった3つ。

  1. 上に並ぶ英語のファイル名は、いったん無視してOK。あれは“置き場所”です。
  2. ページを下にスクロールすると、日本語の説明（README）が出てきます。読むのはそこから。
  3. 迷ったら、まず 「02 緊急・生存辞典」 を開く。今あなたが固まっている症状から引けます。

準備はいりません。まずは、のぞいてみるだけ。
